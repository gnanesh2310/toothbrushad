# Toothbrush Advertisement

A promotional website for a toothbrush brand/product.

## Description
This is a professional advertisement and marketing website for a toothbrush product. It showcases product features, benefits, pricing, and includes a call-to-action for customers.

## Tech Stack
- HTML5
- CSS3
- JavaScript
- Responsive Design

## Features
- Product showcase
- Features and benefits
- Pricing information
- Customer testimonials
- Product images gallery
- Contact/Order form
- Mobile responsive
- Social media integration
- SEO optimized

## Getting Started

### Prerequisites
- A modern web browser

### Installation
1. Clone the repository
   ```bash
   git clone https://github.com/gnanesh2310/toothbrushad.git
   ```
2. Open `index.html` in your browser

## Page Sections
- **Header** - Logo and navigation
- **Hero** - Product showcase and headline
- **Features** - Key product benefits
- **Gallery** - Product images
- **Testimonials** - Customer reviews
- **Pricing** - Product tiers and pricing
- **CTA** - Call-to-action for purchase
- **Footer** - Contact and social links
